# Ceremonies

Structured team meetings. Auto-triggered ceremonies run when their condition matches; manual ceremonies run on request.

## Pre-Ship (auto, after)
- **When:** before user-facing artifacts finalize (releases, public API changes, docs).
- **Facilitator:** Roy
- **Participants:** Rai (RAI review), Fact Checker (verification).

## Design Review (manual)
- **When:** user asks for a design meeting or before a significant architecture change.
- **Facilitator:** Roy
- **Participants:** relevant domain devs.

## Retro (manual)
- **When:** user asks to "run a retro".
- **Facilitator:** Roy
- **Participants:** all active cast agents.
