# Supported Operators

This document defines the operators our runtime (`onnx-runtime-ep-cpu`, `onnx-runtime-ep-cuda`) must implement.

**Scope:**
- ONNX standard ops: opset 17–26 (latest)
- Microsoft contrib ops (`com.microsoft` domain): subset used by Mobius-generated GenAI models

**Non-goal:** We don't implement the full ~180 ONNX op catalog. We target ops used by
transformer/LLM/diffusion/vision models generated by Mobius. For long-tail ops, legacy
plugin EPs (via C ABI bridge) handle them.

---

## 2. Microsoft Contrib Ops (`com.microsoft` domain)

These are the ops Mobius/OGA generates in transformer models. They are **fused** ops
that combine multiple standard ops into a single optimized kernel.

### Tier 1: Critical (Mobius LLM output uses these directly)

| Op | Description | Fuses |
|----|-------------|-------|
| **GroupQueryAttention** | GQA with paged KV cache support | Q×K^T, scale, mask, softmax, ×V, with GQA head sharing. Supports paged block table input. |
| **MultiHeadAttention** | Standard MHA | Same as above but equal Q/K/V heads. |
| **RotaryEmbedding** | RoPE position encoding | cos/sin computation + rotation. Supports interleaved and non-interleaved. |
| **SkipLayerNormalization** | Residual + LayerNorm fused | Add(residual, input) → LayerNorm. One kernel, one memory pass. |
| **SkipSimplifiedLayerNormalization** | Residual + RMSNorm fused | Add(residual, input) → RMSNorm (no mean subtraction). |
| **SimplifiedLayerNormalization** | RMSNorm | Norm without mean subtraction (Llama-style). |
| **MatMulNBits** | INT4 quantized MatMul | Weight is INT4 packed. Dequant + GEMM fused. |
| **BiasAdd** | Add bias to tensor | Simple but fused for scheduling. |
| **FastGelu** | Approximated GELU | tanh approximation, faster than exact. |
| **BiasGelu** | Bias + GELU fused | MatMul output → add bias → GELU in one kernel. |
| **BiasSplitGelu** | Bias + Split + GELU (for GLU variants) | Used in SwiGLU/GeGLU architectures. |
| **QuickGelu** | x × sigmoid(α × x) | Used in some CLIP/Phi models. |
| **GatedRelativePositionBias** | Relative position bias with gating | |

### Tier 2: Important (Mobius multimodal/diffusion)

| Op | Description | Notes |
|----|-------------|-------|
| **SkipGroupNorm** | Residual + GroupNorm fused | Diffusion U-Net blocks. |
| **NhwcConv** | Conv in NHWC layout | Avoids layout transpose for channels-last models. |
| **FusedConv** | Conv + Bias + Activation fused | |
| **QOrderedMatMul** | Quantized ordered MatMul | INT8 with specific memory ordering. |
| **QAttention** | Quantized attention | INT8 Q/K/V. |
| **DecoderMaskedMultiHeadAttention** | Decode-only MHA with causal mask baked in | Optimized for single-token decode step. |
| **DecoderMaskedSelfAttention** | Self-attention for decode | |
| **PackedAttention** | Variable-length packed sequences | No padding, batch packed together. |
| **PackedMultiHeadAttention** | Packed MHA | |
| **PagedAttention** | Paged KV cache attention | Block table driven, vLLM-style. |
| **SparseAttention** | Sparse attention patterns | Sliding window, local+global. |

### Tier 3: Utility (preprocessing, sampling)

| Op | Description | Notes |
|----|-------------|-------|
| **Sampling** | Top-k/top-p/temperature sampling | OGA uses this. |
| **BeamSearch** | Beam search decoding | |
| **GreedySearch** | Greedy decoding | |
| **EmbedLayerNormalization** | Embedding + LayerNorm fused | BERT-style. |
| **Attention** (legacy) | Original fused attention | Older models; prefer GQA/MHA. |

## 3. Implementation Priority Matrix

```
Phase 1 (CPU EP — basic inference):
  Standard Tier 1 (core math + reshape + gather + layernorm + softmax)
  Contrib: SimplifiedLayerNormalization, SkipLayerNormalization, FastGelu, BiasAdd

Phase 2 (CUDA EP — LLM decode):
  Standard Tier 1 complete
  Contrib: GroupQueryAttention, RotaryEmbedding, MatMulNBits,
           SkipSimplifiedLayerNormalization, BiasGelu, BiasSplitGelu

Phase 3 (Full LLM support):
  Standard Tier 2 (quantization ops, einsum, trilu, range)
  Contrib: MultiHeadAttention, PagedAttention, SparseAttention,
           DecoderMaskedMultiHeadAttention, PackedAttention

Phase 4 (Multimodal + Diffusion):
  Standard Tier 2 complete (conv, resize, pool, batchnorm)
  Contrib: SkipGroupNorm, NhwcConv, FusedConv

Phase 5 (Full coverage):
  Standard Tier 3
  Contrib Tier 3
```

## 5. Minimum Opset Version Policy

We support **opset 17 as minimum** (introduced LayerNormalization as standard op).
Models with opset < 17 should be upgraded with `onnx.version_converter` before use.

For each op, we implement the **latest semantics** and provide backward-compat shims
for older opset versions where behavior differs (e.g. Reshape axis handling, Reduce* keepdims default).

## 6. Custom Ops Extension Point

Users can register additional ops via the C ABI custom op interface (same mechanism as ORT Extensions):

```rust
pub trait CustomOp: Send + Sync {
    fn name(&self) -> &str;
    fn domain(&self) -> &str;
    fn since_version(&self) -> u64;
    fn create_kernel(&self, node: &Node) -> Result<Box<dyn Kernel>>;
}

impl InferenceSession {
    /// Register custom ops from a shared library.
    pub fn register_custom_ops(path: &Path) -> Result<()>;
}
```
